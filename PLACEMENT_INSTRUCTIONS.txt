FINORA - Phase 3 File Placement Guide
=======================================

1. dashboard/services.py              -> finora/dashboard/services.py (NEW file)
2. dashboard/views.py                 -> finora/dashboard/views.py (overwrite)
3. dashboard/templates/dashboard/home.html -> finora/dashboard/templates/dashboard/home.html (overwrite)
4. static/js/dashboard.js             -> finora/static/js/dashboard.js (NEW file)

BEFORE RUNNING: install python-dateutil (used for safe month arithmetic)

    pip install python-dateutil

No new migrations needed in this phase (no model changes) — just:

    python manage.py runserver

Then test using the checklist provided in chat.

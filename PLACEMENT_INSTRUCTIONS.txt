WEALTHORA - Phase 10 File Placement Guide
===========================================

1. templates/404.html -> wealthora/templates/404.html (NEW - project-level templates folder, NOT inside an app)
2. templates/500.html -> wealthora/templates/500.html (NEW - same folder)

3. .gitignore -> wealthora/.gitignore (OVERWRITE your existing one with this finalized version)

4. SECURITY_CHECKLIST.txt -> keep anywhere handy, e.g. wealthora/SECURITY_CHECKLIST.txt
   This is for YOU to read and manually verify -- not code to run.

5. DEPLOYMENT_NOTES.md -> wealthora/DEPLOYMENT_NOTES.md
   Reference doc for when you're ready to deploy live.

NOTE: 404.html and 500.html only activate when DEBUG=False in settings.py.
With DEBUG=True (your current local dev setting), Django always shows its
own detailed debug error pages instead -- that's normal and expected.
To actually SEE your custom 404 page while testing locally, you can
temporarily set DEBUG=False and ALLOWED_HOSTS=['127.0.0.1', 'localhost']
in your .env/settings, then visit a nonsense URL like /this-does-not-exist/.
Remember to set DEBUG back to True afterward for continued local development.
